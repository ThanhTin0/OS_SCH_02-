## 📝 Mô tả dự án
Ứng dụng giao diện đồ họa (GUI) sử dụng Tkinter để mô phỏng và trực quan hóa hoạt động của 2 thuật toán lập lịch CPU cơ bản:
* **FCFS** (First-Come, First-Served)
* **Priority** (Non-preemptive - Độc quyền)
